# 贡献指南

感谢你对本项目感兴趣！

## 开发设置

1. Fork 本仓库
2. 克隆你的 fork

```bash
git clone https://github.com/YOUR_USERNAME/vite-mpa.git
cd vite-mpa
```

3. 安装依赖

```bash
pnpm install
```

4. 进行你的修改

## 提交规范

使用语义化的提交信息：

- `feat: 新功能`
- `fix: 修复bug`
- `docs: 文档更新`
- `style: 代码格式（不影响代码运行的变动）`
- `refactor: 重构`
- `perf: 性能优化`
- `test: 增加测试`
- `chore: 构建过程或辅助工具的变动`

## Pull Request

1. 确保你的代码符合项目的代码规范
2. 更新相关文档
3. 提交 Pull Request，清晰描述你的修改

## 问题反馈

如果你发现了 bug 或有新功能建议，请创建一个 issue。
